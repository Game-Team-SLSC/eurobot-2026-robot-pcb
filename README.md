This repo contains the source files for the PCB for the 2026 robot of Game Team SLSC

# Setup

In order to work with this repo, you'll need the following prerequisites :

* [Kicad v9.0.6 compatible](https://www.kicad.org/download/)
* [Espressif Kicad library 3.0.2 compatible](https://github.com/espressif/kicad-libraries/releases/tag/3.0.2)

Libraries are project-contained (/libs), so you're fine using the default sym/footprints table
